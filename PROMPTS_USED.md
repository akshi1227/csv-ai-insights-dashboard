# Prompts Used

## Prompt 1: Backend Structure
> "Generate a project structure for a Node.js Express server that handles CSV file uploads using multer and has a route for generating insights using the Google Gemini API. Include error handling and a simple folder structure."

## Prompt 2: React File Upload Component
> "Create a React component using functional hooks that allows users to drag and drop a CSV file or click to select one. Display the file name upon selection and show a progress bar during upload. Use standard CSS for styling."

## Prompt 3: AI Insights Prompt Design
> "I have a dataset summary containing column names, data types, and the first 5 rows of data. Write a system prompt for an LLM that asks it to analyze this summary and identify trends, outliers, and potential quality issues. The output should be in Markdown format."

## Prompt 4: API Integration Debugging
> "My fetch request to the backend is failing with a CORS error. How do I configure the `cors` middleware in Express to allow requests from my request frontend running on verify localhost:5173?"
